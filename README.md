
# Legal AI assistant

A Legal AI assistant mainly used for predicting case strength, Contract analysis, Dispute Resolution and drafting new contract. 

## Features

- Predictive Analysis.
- Discussions.
- Dispute resolution.
- Contract analysis and drafting.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Srimadhavn/LexIntellect.git
```

Go to the project directory

```bash
  cd LexIntellect
```

Install dependencies

```bash
  cd server
```
*// Create venv for running pip*
```
  pip install -r requirements.txt
```

*// Open new terminal*
```  
   cd client
   npm install
```

Start the server

*// flask server*
```bash
   python app.py
```

*//Next.js server*

```
   npm run dev

```
 Visit http://localhost:3000/ in browser.



## Tech Stack

**Client:** Next, TailwindCSS

**Server:** Flask
